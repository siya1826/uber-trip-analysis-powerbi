# Uber Trip Analysis Dashboard (Power BI)

## Project Overview
This project is a Power BI dashboard created to analyze Uber trip data, including total trips, revenue, trip distance, trip time, and city-wise performance.

## Problem Statement
The objective of this project is to analyze Uber trip data to understand:
- Total number of trips
- Total revenue
- Trip distance and trip time
- Peak booking time
- City-wise trip performance
- Trip trends over time

## Tools Used
- Power BI
- Power Query (Data Cleaning)
- DAX (Measures)
- Data Visualization

## Process
1. Imported dataset into Power BI
2. Cleaned data using Power Query
3. Created date table for time analysis
4. Created DAX measures:
   - Total Trips
   - Total Revenue
   - Average Trip Distance
   - Average Trip Time
5. Created visualizations:
   - Trips by Time
   - Trips by City
   - Revenue Analysis
   - Trip Trends
6. Added filters for Date and City

## Key Insights
- Total Trips: 103K
- Total Revenue: $1.6M
- Average Trip Distance: 15 km
- Peak trip time in evening hours
- Some cities generate higher revenue than others
- Trips increase on weekends

## Conclusion
This dashboard helps Uber understand trip trends, peak hours, and city performance, which helps in driver allocation and pricing strategy.
